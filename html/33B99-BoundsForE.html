bounds for e


If n and m are positive integers and n>m, we have the following inequality:

(1+1n)n<nn+1⁢(1+1m)m+1

Taking the limit as n→∞, we obtain an upper bound for e. Combining this with the fact that the (1+1/n)n is an increasing sequence, we have the following bounds for e:

(1+1m)m<e<(1+1m)m+1

This can be used to show that e is not an integer – if we take m=5, we obtain 2.48832<e<2.985984, for instance.
