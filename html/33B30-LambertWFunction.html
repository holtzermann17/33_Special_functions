Lambert W function


Lambert’s W function is the inverse of the function f:ℂ→ℂ given by f⁢(x):=x⁢ex. That is, W⁢(x) is the complex valued function that satisfies

W⁢(x)⁢eW⁢(x)=x,

for all x∈ℂ. In practice the definition of W⁢(x) requires a branch cut, which is usually taken along the negative real axis. Lambert’s W function is sometimes also called product log function.

This function allow us to solve the functional equation

g⁢(x)g⁢(x)=x

since

g⁢(x)=eW⁢(ln⁡(x)).
